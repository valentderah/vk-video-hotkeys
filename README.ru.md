  <h2>Горячие клавиши VK Video <img src="public/images/icons/icon-32.png" width="20px"></h2>
  <p>Это расширение возвращает привычное управление плеером как на YouTube с помощью клавиш JKL.</p>
</div>

## Возможности
- **Привычное управление**: J, K, L для навигации.
- **Контроль скорости**: Изменение скорости клавишами `<` и `>`.
- **UI управление**: Дополнительные кнопки перемотки прямо в интерфейсе плеера.
- **Удержание для ускорения**: Удерживайте Пробел для временного ускорения видео до 2x.
- **Перемотка цифрами**: Цифры от 0 до 9 для перехода к процентам видео.
- **Настройка**: Можно переназначить клавиши под себя в меню расширения.

## Управление по умолчанию

| Клавиша           | Действие                            |
|-------------------|-------------------------------------|
| J (О)             | Назад на 15 секунд                  |
| K (Л)             | Пауза / Проигрывание                |
| L (Д)             | Вперед на 15 секунд                 |
| 1-9               | Перемотка видео в процентах         |
| C (С)             | Включить/Выключить субтитры         |
| T (Е)             | Включить/Выключить режим кинотеатра |
| \<                | Уменьшить скорость воспроизведения  |
| \>                | Увеличить скорость воспроизведения  |
| Удержание пробела | Ускорение видео до (2x)             |

<summary>Ручная установка</summary>

1. Скачайте последний релиз или склонируйте репозиторий.
2. Откройте страницу расширений (`chrome://extensions`).
3. Включите **Режим разработчика** (правый верхний угол).
4. Нажмите **Загрузить распакованное расширение** (левый верхний угол).
5. Выберите папку с манифестом расширения.
